Thyssen CLI Tool

Efficient system monitoring and JSON file management

About This Project

This is a command-line interface (CLI) tool built using Python and Click. It offers system administrators an intuitive way to monitor CPU, memory, disk usage, and manage JSON files.

Features

  • Monitor CPU, memory, and disk usage.
  • Manage JSON files: display, copy, modify, and delete them.
  • Simple command-line operations.
